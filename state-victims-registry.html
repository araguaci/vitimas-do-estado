🤝 Como Contribuir
  • 1
    Fork o repositório no GitHub. Todos os dados vivem em arquivos JSON versionados — sem banco de dados central.
  • 2
    Crie um arquivo JSON na pasta cases/ seguindo o schema em SCHEMA.md. Fontes obrigatórias com link de arquivo.
  • 3
    Abra um Pull Request. O CI valida o schema automaticamente. Um segundo contribuidor humano revisa as fontes antes do merge.
  • 4
    Sem anonimato editorial — o contribuidor assina o registro. Isso não é para covardes: é um ato de responsabilidade.
