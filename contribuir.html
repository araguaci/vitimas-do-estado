Como contribuir

Fluxo editorial e requisitos técnicos

github.com/araguaci/vitimas-do-estado · Fork · Issues · Pull requests

Adicionar caso

Faça fork do repositório, crie um ficheiro JSON na pasta cases/ com fontes verificáveis e abra um pull request.

Cada registo deve poder ser verificado por terceiros — links primários, arquivos ou documentos públicos.

Schema JSON

O schema está descrito em SCHEMA.md no repositório. Campos obrigatórios incluem identificador, slug, país, narrativa e lista de fontes com URL.

Diretrizes

  • Sem anonimato editorial — o contribuidor assina o registo.
  • Sem acusações sem fonte: cada afirmação grave deve apontar para evidência.
  • Respeito às vítimas e famílias — linguagem factual, sem sensacionalismo.

Abrir issue para questões editoriais

Adicionar país

Abra uma discussão (issue) ou um PR com contexto jurídico e fontes sobre o sistema estatal do país. O manifest registry.json lista os datasets por região.
